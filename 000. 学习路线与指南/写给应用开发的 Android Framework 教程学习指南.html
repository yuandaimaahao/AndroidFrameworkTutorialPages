    写给应用开发的 Android Framework 教程学习指南

    阿豪 7/5/2023

    本文是系列教程 “写给应用开发的 Android Framework 教程” 的学习指南，随教程内容持续更新中。

    下图是教程的整体大概，教程的内容会围绕大纲展开：

    # 1. 基础准备篇

    每个人根据自己情况学习基础部分，建议先学后面的，遇到不懂的再回头学习/复习基础部分：

    • 熟悉 Linux 的基本使用
    • C/C++ 花几天时间把语法快速过一遍，有个印象就行，不用精通熟练
    • 看看 JNI 编程，能自己写点简单 demo

    # 2. 玩转 AOSP 篇

    玩转 AOSP 篇是教程的第一个专题，主要内容是下载好 AOSP 源码，修改源码，给源码添加一些功能。完成该专题的学习后能对系统开发有基本的了解。

    该专题主要包含了以下内容：

    • 学习本专题需要熟悉 Linux 的基本操作，具体可以参考Ubuntu 使用快速入门 (opens new window)
    • 首先我们应该参考AOSP 极速上手 (opens new window)这篇文章，下载编译源码，把虚拟机跑起来。
    • 接着学习Android 系统开发工具推荐 (opens new window),熟悉常用的开发工具。
    • 然后我们可以学习添加 Product (opens new window)，了解给系统添加功能的准备工作。
    • 接着我们可以看看添加可执行程序 (opens new window)、添加 C/C++、Java 库 (opens new window)、添加配置文件与删除已有模块 (opens new window)以及添加开机自启动 Shell 脚本 (opens new window),来了解如何在系统源码中添加一些基本功能。
    • 接下来可以通过系统 App 源码添加 (opens new window)和使用 Android Studio 开发系统 App (opens new window)来入门系统 App 的开发
    • SeAndroid 在系统开发中随时会遇到，可以通过SeAndroid 使用极速上手 (opens new window) 来入门 SeAndroid 的使用
    • 最后，我们可以通过如何阅读 Android 系统源码 —— C/C++ 篇 (opens new window)和如何阅读 Android 系统源码 —— Java 篇 (opens new window) 来入门系统源码的阅读。

    # 3. 学穿 Binder 篇

    Binder 涉及的知识面很广，建议根据自身情况将必要的基础知识过一遍：

    • AOSP 上手指南
    • Linux 内核下载编译运行
    • Linux 驱动开发入门
    • Linux 内核常用数据结构
    • 虚拟内存
    • Linux 文件访问接口
    • JNI 编程入门

    这部分内容在学习 Binder 的预备知识 (opens new window) 和基础篇中都有分享。

    然后我们通过 Binder 基本原理 (opens new window) 了解 Binder。

    接着我们参考源码中的 bctest 与 servicemanager 完成一个 c 语言的 Binder 示例程序：

    • Binder 程序示例之 C 语言篇 (opens new window)
    • Binder 服务注册过程情景分析之 C 语言篇 (opens new window)
    • Binder 服务获取与使用过程情景分析之C语言篇 (opens new window)
